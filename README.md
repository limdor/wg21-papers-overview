# About this repo

Participating in the C++ committee is time consuming, and for a lot of us, it is not our primary work.
Because of that, I do not always have as much time as I would like to stay up-to-date and read all new papers. To use the time I has as efficient as possible, I decided to create this tool to generate summaries to know what has been going on between plenaries.

The published summaries can be found as gists:

- [Papers between Kona 11-2025 and Croydon 03-2026](https://gist.github.com/limdor/476edd5844ac24fe64ba5773e2c5365d#file-updatesfromkonatocroydon-md)
- [Papers between Sofia 06-2025 and Kona 11-2025](https://gist.github.com/limdor/5795b710d929a663b2af7c27f16aa175#file-updatesfromsofiatokona-md)
- [Papers between Hagenberg 02-2025 and Sofia 06-2025](https://gist.github.com/limdor/e22a5316bfa9c35f7ed425a900d6a37b#file-updatesfromhagenbergtosofia-md)
